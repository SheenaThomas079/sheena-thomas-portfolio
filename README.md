# Sheena Thomas - Portfolio

A simple, fast, and beautifully interactive modern portfolio website.

## 📌 Tech Stack
- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- **Extras:** AOS (Animate on Scroll), Typed.js, FontAwesome.

## ⚡ Features
- Single HTML file using modern CDN integrations.
- Tailwind CSS styling with minimal custom styling overrides.
- No bulky node modules, frameworks, or webpack configurations required.
- Dynamically generates Projects and Skills section data via simple local JS objects.
- Easily deployable offline or simply by dragging into the browser.
